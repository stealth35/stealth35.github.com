Stealth35 PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover stealth35.github.com/pear
Listing available packages: 
pear remote-list -c stealth35
Installing a package: 
pear install stealth35/package_name
Installing a specific version/stability: 
pear install stealth35/package_name-1.0.0
pear install stealth35/package_name-beta
Receiving updates via a feed: 
http://stealth35.github.com/pear/feed.xml
